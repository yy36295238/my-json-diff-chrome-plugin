基础字段对比

左侧JSON

{
  "name": "张三",
  "age": 25,
  "city": "北京",
  "active": true
}

右侧JSON

{
  "name": "张三",
  "age": 26,
  "city": "上海",
  "active": true,
  "email": "zhang@example.com"
}

预期差异

age: 修改 city: 修改 email: 新增

嵌套对象对比

左侧JSON

{
  "user": {
    "name": "李四",
    "profile": {
      "age": 30,
      "city": "北京"
    }
  },
  "settings": {
    "theme": "dark",
    "lang": "zh"
  }
}

右侧JSON

{
  "user": {
    "name": "李四",
    "profile": {
      "age": 31,
      "city": "上海",
      "country": "中国"
    }
  },
  "config": {
    "theme": "light"
  }
}

预期差异

profile.age: 修改 profile.city: 修改 profile.country: 新增 settings: 删除 config: 新增

数组结构对比

左侧JSON

{
  "fruits": ["apple", "banana"],
  "colors": ["red", "green", "blue"],
  "count": 5
}

右侧JSON

{
  "fruits": ["apple", "orange", "banana", "grape"],
  "colors": ["red", "yellow", "blue"],
  "count": 7,
  "total": 12
}

预期差异

fruits[1]: 修改 fruits[2]: 新增 fruits[3]: 新增 colors[1]: 修改 count: 修改 total: 新增

复杂混合结构

左侧JSON

{
  "users": [
    {"id": 1, "name": "Alice"},
    {"id": 2, "name": "Bob"}
  ],
  "meta": {
    "version": "1.0",
    "enabled": true
  }
}

右侧JSON

{
  "users": [
    {"id": 1, "name": "Alice", "email": "alice@example.com"},
    {"id": 3, "name": "Charlie"}
  ],
  "meta": {
    "version": "2.0",
    "enabled": false,
    "updated": "2024-01-01"
  }
}

预期差异

users[0].email: 新增 users[1]: 修改 meta.version: 修改 meta.enabled: 修改 meta.updated: 新增
