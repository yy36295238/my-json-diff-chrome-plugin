# JSON Editor Pro (JSON 工具)

JSON Editor Pro 是一款专为开发者和数据分析师设计的现代化 Chrome 扩展。它提供了一套功能强大的工具，用于在浏览器中直接格式化、校验、比较、转换和生成 JSON 数据。

## ✨ 核心功能

### 1. 格式化与校验 (Formatter & Validator)
- **实时校验**: 输入时即时进行语法检查。
- **美化/压缩**: 一键格式化（美化）或压缩 JSON 数据。
- **深度递归解析**: 自动检测并解析嵌套的 JSON 字符串（即使包含转义字符）。
- **转义/去转义**: 轻松处理 JSON 字符串中的转义字符（如 `"`、`\n` 等）。
- **树状视图预览**: 可交互的树状视图，支持折叠/展开节点，便于浏览复杂结构。
- **JSONPath 支持**: 点击预览中的任意键即可生成对应的 JSONPath 路径。

### 2. JSON 比较 (Diff)
- **可视化差异**: 左右分栏直观对比两个 JSON 对象。
- **智能高亮**: 使用不同颜色高亮显示新增（绿色）、删除（红色）和修改（琥珀色）的内容。
- **结构感知**: 忽略空白字符和键值顺序的差异，专注于数据结构的变化。

### 3. 模拟数据生成 (Mock Data Generator)
- **多种模板**: 内置用户、商品、订单、地址、公司、文章、API 响应、系统配置等多种常用数据模板。
- **多语言支持**: 支持生成中文或英文的模拟数据。
- **灵活配置**: 支持自定义生成数量，支持批量生成数组或单对象。
- **一键应用**: 生成的数据可直接应用到编辑器中进行后续操作。

### 4. 转换器 (Converter)
- **JSON 转 Java**: 将 JSON 自动转换为 Java 实体类（支持 `@Data` Lombok 注解）。
- **JSON 转 SQL**: 将 JSON 数组转换为 SQL `INSERT` 语句。
- **SQL 转 JSON**: 将标准的 SQL `INSERT` 语句解析回 JSON 格式。

### 5. 多面板编辑器 (Multi-Pane Editor)
- **分屏视图**: 最多支持创建 5 个可调整大小的独立面板，同时处理多个 JSON 片段。
- **独立操作**: 每个面板都可以独立进行格式化、编辑和美化操作。
- **自定义标题**: 可重命名面板以便于组织工作区。
- **状态持久化**: 自动保存布局和内容，防止数据丢失。

### 6. 现代 UI/UX
- **主题支持**: 支持浅色（Light）和深色（Dark）模式切换。
- **响应式布局**: 可拖拽调整面板大小，适应不同屏幕。
- **隐私优先**: 所有数据处理均在本地浏览器中完成，**不会**向任何服务器发送数据。

## 🚀 安装指南

1.  **克隆仓库:**
    ```bash
    git clone <repository-url>
    cd my-json-diff-chrome-plugin
    ```

2.  **加载到 Chrome:**
    *   打开 Chrome 浏览器，并在地址栏输入 `chrome://extensions/`。
    *   开启右上角的 **开发者模式 (Developer mode)**。
    *   点击 **加载已解压的扩展程序 (Load unpacked)**。
    *   选择项目目录下的 `chrome-extension` 文件夹。

3.  **固定扩展:**
    *   点击 Chrome 工具栏上的拼图图标。
    *   找到 **JSON工具** 并点击图钉图标将其固定，方便快速访问。

## 🛠 开发说明

本项目结构简洁，无需复杂的构建步骤（无需 Webpack/Rollup），使用原生的 ES Modules。

*   `chrome-extension/`: 扩展的根目录。
    *   `manifest.json`: 扩展配置文件 (Manifest V3)。
    *   `index.html`: 主 UI 入口。
    *   `styles-simple.css`: 核心样式变量和 CSS。
    *   `js/`: 应用程序逻辑。
        *   `app.js`: 主入口和状态管理。
        *   `managers/`: 特定功能模块 (Formatter, Compare, Layout, Converter, Generator)。
        *   `utils.js`: 通用工具函数。
