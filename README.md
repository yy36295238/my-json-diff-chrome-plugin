# JSON工具 - 专业级JSON处理平台

一个功能全面的JSON处理工具，提供格式化、对比、生成、验证、转换和分析等功能。

## 🚀 主要特性

### 📝 JSON格式化器
- **美化JSON**: 添加缩进和换行，提升可读性
- **压缩JSON**: 移除多余空格，减小文件大小
- **实时验证**: 输入时即时检测JSON语法错误
- **语法高亮**: 不同元素类型的颜色标记

### 🔍 JSON对比工具
- **行级差异高亮**: 直接在输入框中标记差异
- **并排对比视图**: 完整的差异分析展示
- **差异统计**: 显示新增、删除、修改的行数
- **交换功能**: 一键交换左右对比内容

### 🎲 数据生成器
- **8种预设模板**: 用户、商品、订单、地址等
- **多语言支持**: 中文、英文、日文
- **自定义配置**: 可调整生成数量和格式
- **随机数据**: 真实感的模拟数据

### 🔄 格式转换器
- **多格式支持**: JSON ↔ XML/YAML/CSV/URL参数
- **双向转换**: 支持任意格式间的相互转换
- **智能识别**: 自动检测输入格式
- **美化输出**: 可选择格式化输出结果

### ✅ JSON验证器
- **详细错误信息**: 精确的错误位置和修复建议
- **JSONPath查询**: 支持标准JSONPath语法
- **数据统计**: 文件大小、行数、类型分布
- **结构分析**: 层级深度、节点数量统计

### 📊 数据分析器
- **结构分析**: 深度、键值、类型分布统计
- **性能指标**: 解析时间、内存占用估算
- **树形展示**: 可视化JSON结构层级
- **质量检查**: 重复值、异常数据检测

## 🎯 使用示例

```json
// 输入压缩的JSON

{
  "name": "张三",
  "age": 25,
  "skills": [
```

## 🐛 常见问题

### Q: 为什么我的JSON显示验证失败？
A: 请检查：
- 字符串是否使用双引号
- 是否有多余的逗号
- 括号是否正确匹配

### Q: 大文件处理很慢怎么办？
A: 对于大文件建议：
- 使用压缩功能减少文件大小
- 分段处理超大文件
- 关闭实时验证功能

### Q: 如何保存我的工作？
A: 工具提供多种保存方式：
- 使用导出功能保存文件
- 历史记录自动保存
- 浏览器本地存储
