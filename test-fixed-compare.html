基本字段修改测试

左侧JSON

{
  "name": "张三",
  "age": 25,
  "city": "北京"
}

右侧JSON

{
  "name": "张三",
  "age": 26,
  "city": "上海",
  "email": "zhang@example.com"
}

预期结果

  • name字段保持不变（无高亮）
  • age字段显示修改（黄色高亮）
  • city字段显示修改（黄色高亮）
  • 右侧email字段显示新增（绿色高亮）
  • 左右两侧行数对齐，无重复数据

数组长度不同测试
